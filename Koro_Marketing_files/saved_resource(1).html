Cách thiết lập & sử dụng Claude Project cho Koro
Skill là gì và cách dùng cho đội Koro?
Skill là file hướng dẫn (.md) dạy Claude xử lý đúng một loại yêu cầu — đúng framework, đúng định dạng kết quả phù hợp EdTech tiếng Hàn. Tải lên Claude Project một lần, gõ bằng tiếng Việt, Claude tự nhận và xuất file hoàn chỉnh.
BƯỚC 1 — TẢI FILE
Tải 13 file .md
Nhấn "Tải .md" từng skill hoặc tải tất cả. File định dạng Markdown, mở được bằng Notepad hoặc VS Code.
BƯỚC 2 — TẠO PROJECT
Tạo Claude Project riêng
claude.ai → Projects → New. Vào Project Knowledge tải toàn bộ file .md. Mỗi vị trí nên có Project riêng.
BƯỚC 3 — DÙNG NGAY
Gõ tiếng Việt là được
"Lập kế hoạch marketing cho Koro tháng 3" hoặc "Viết script TikTok về bảng chữ cái Hangul" — Claude tự nhận skill phù hợp.
